# ZION Runner 3.0 — Stage 3

Stage 3 is the final, complete build of the modular ZION Runner 3.0 game.

## Included

- all six UPL transport missions from Stage 2
- final Quantum Jammer survival showdown
- keys, trusted nodes, speed tunnels and secure shields
- persistent keys earned after every run
- three permanent upgrades with three levels each
- five unlockable achievements
- delivery count, best score and upgrade profile
- combo, hit, route and key statistics
- keyboard, swipe and mobile controls
- pause/resume and visibility auto-pause
- synthesized music and gameplay sounds
- local save data with no account or network dependency

## Permanent upgrades

- **Packet Armor** — adds one starting shield per level
- **Key Magnet** — increases collection range
- **Trusted Relay** — increases all score gains

## Run

ES modules require a local HTTP server:

```bash
python3 -m http.server 8080
```

Open:

```text
http://localhost:8080
```

## Controls

- `A` / `D` or `←` / `→` — move
- `W`, `↑` or `Space` — jump / double jump
- `P` or `Escape` — pause
- mobile — on-screen controls or upward swipe
